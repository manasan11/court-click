# Court Click — Admin Dashboard

A Next.js admin dashboard for managing CTC (Certified True Copy) orders with a deep plum/maroon design system.

## Tech Stack

- **Next.js** (App Router)
- **React** 19
- **Ant Design** 6
- **Lucide React** (icons)
- **TypeScript**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
